 DoAgro 

 - Architecture
 
It`s a mobile application wich runs successfully on Android and IOS operating systems.
It is created in .NET Framework using Xamarin.Forms cross-platform framework in Visual Studio 2019 environment.
For back-end I used C# and for front-end XAML.
As cloud database, the application is connected with Firebase platform for users management. For local database I used SQLite.

 - Utility
 
 The main and the only language is Romanian.
 The application has multiple fucntionalities wich can help the users to improve their knowledge in agriculture and planting specific crops.
